# حساب تجريبي للتطوير

## معلومات الحساب التجريبي

تم إضافة دعم لحساب تجريبي ثابت لتسهيل عملية التطوير والاختبار.

### بيانات الحساب

- **رقم الهاتف**: `07761763665`
- **رمز OTP**: `123456`
- **كلمة المرور**: `test123456` (للدخول بكلمة المرور)

## كيفية الإعداد

### 1. إعداد Backend

قم بإضافة المتغيرات التالية إلى ملف `.env` في مجلد `backend`:

```env
TEST_PHONE_NUMBER=07761763665
TEST_OTP_CODE=123456
```

### 2. إنشاء المستخدم التجريبي في قاعدة البيانات

قم بتشغيل السكريبت التالي لإنشاء المستخدم التجريبي:

```bash
cd backend
npm run create-test-user
```

أو:

```bash
cd backend
node src/scripts/createTestUser.js
```

### 3. إعداد Frontend

تم إضافة المتغيرات التالية إلى ملف `.env.local`:

```env
EXPO_PUBLIC_TEST_PHONE=07761763665
EXPO_PUBLIC_TEST_OTP=123456
```

## كيفية الاستخدام

### تسجيل الدخول

1. افتح التطبيق
2. اختر "تسجيل الدخول"
3. أدخل رقم الهاتف: `07761763665`
4. اضغط على "إرسال رمز التحقق"
5. أدخل رمز OTP: `123456`
6. اضغط على "تسجيل الدخول"

### التسجيل (إذا لم يكن الحساب موجوداً)

1. افتح التطبيق
2. اختر "إنشاء حساب جديد"
3. أدخل رقم الهاتف: `07761763665`
4. اضغط على "إرسال رمز التحقق"
5. أدخل رمز OTP: `123456`
6. أكمل باقي البيانات
7. اضغط على "إنشاء الحساب"

## كيف يعمل؟

### في Backend

تم تعديل ملف `backend/src/config/otp.js` ليتحقق من رقم الهاتف:

- إذا كان الرقم هو `07761763665`، يتم إرجاع `orderId` تجريبي بدون إرسال رسالة WhatsApp حقيقية
- عند التحقق من OTP، إذا كان `orderId` تجريبياً، يتم قبول الرمز `123456` فقط

### خدمة OTP

- يتم استخدام **OTP.dev** لإرسال رموز التحقق
- القناة المستخدمة: **WhatsApp** (وليس SMS)
- للأرقام الحقيقية، سيتم إرسال رمز التحقق عبر رسالة WhatsApp

### الأمان

⚠️ **تحذير**: هذه الميزة للتطوير فقط!

- لا تستخدم هذا الحساب في بيئة الإنتاج
- تأكد من حذف أو تعطيل هذه الميزة قبل النشر
- يمكنك تعطيل الحساب التجريبي بحذف المتغيرات `TEST_PHONE_NUMBER` و `TEST_OTP_CODE` من ملف `.env`

## ملاحظات

- الحساب التجريبي يعمل مع كل من تسجيل الدخول والتسجيل
- لا يتم إرسال رسالة WhatsApp حقيقية عند استخدام الرقم التجريبي
- للأرقام الحقيقية، سيتم إرسال رمز OTP عبر WhatsApp
- يمكنك تغيير الرقم ورمز OTP من ملف `.env`
- السكريبت `createTestUser.js` يتحقق من وجود المستخدم قبل إنشائه
